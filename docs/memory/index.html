ConducIA

Intelligence for Urban Mobility

ConducIA es una solución de Big Data diseñada para transformar el transporte en Nueva York. Analizamos millones de registros de Yellow Taxis y VTCs (Uber/Lyft) para predecir precios y tiempos, ayudando a las empresas a optimizar sus costes logísticos frente a la tarifa dinámica.
